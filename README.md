# 高校社团管理系统 java200

## 项目概述

高校社团管理系统是一款旨在帮助高校管理和便捷化社团运作的软件。该系统基于前后端分离的模式，使用Java和SpringBoot进行后端开发，Vue.js进行前端开发。

## 技术栈

- 后端：Java, SpringBoot
- 前端：Vue.js
- 数据库：Mysql
- 开发环境：IDEA, JDK 1.8
- 系统环境：Windows, Mac
